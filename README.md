# 🛒 Grocery Billing System (Python OOP)

A console-based Grocery Billing System developed using Python and Object-Oriented Programming (OOP). This project simulates a supermarket billing system where users can add products, manage a shopping cart, generate bills, and store purchase history.

## 🚀 Features

- Display available products
- Add items to cart
- Remove items from cart
- Update item quantity
- View shopping cart
- Generate final bill
- Automatic discount calculation
- GST calculation
- Purchase history saved to a text file
- Admin settings to update store name and GST rate

## 🛠️ Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- File Handling
- Exception Handling
- Dictionary Data Structure
- Datetime Module

## 📚 OOP Concepts Used

- Classes & Objects
- Constructor (`__init__`)
- Instance Variables
- Class Variables
- Instance Methods
- Class Methods
- Static Methods
- Magic Method (`__str__`)
- Basic Encapsulation
